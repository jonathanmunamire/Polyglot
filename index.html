Programs

12months Class Program

If you want to learn with a whole class, evolve and sharpen you skills with a group of speakers from many countries, and you want to increase you language interaction skills, this is a program for you

12months Personal Program

If you want to learn with a personal teacher, one-on-one tutor and you want more orientation, personal attention and private instruction through the program, This is the program for you

Children Scheduled Program

You have children and want them to learn languages; This is a program for you! We have certified tutors, able to teach children of any age either in a group program or a one-on-one program with your children

Weekly Home Program

If you are unable or you don't want to leave your home, and if you want to have a tutor coming to your place weekly and helping you complete your program from home, this is a program for you

Online Program

If you want to learn from home and you don't want a personal tutor to come at your place, this is a program for you! You can still choose which of the online class program or the online personal program fits your wishes better
